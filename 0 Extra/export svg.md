
# Export SVG

- Export Svg
-   Select Format:  **SVG (**svg**)**

Use the following settings:

-   Styling:  **Internal CSS**
-   Font:  **Convert To Outline**
-   Images:  **Preserve**
-   Decimal:  **4**
-   Minify:  **Checked**
-   Responsive:  **Unchecked**
